# 미션 타입 설정 가이드

미션은 미션 타입 + 라운드 (회차) + 달성 목표 설정에 따라 동작 방식과 보상 지급 구조가 달라집니다.\
아래 예시를 참고하여 목적에 맞는 미션 타입을 선택해 주세요.

## SINGLE - 단일 목표 미션

* 단일 목표를 달성하면 보상이 지급됩니다.
* 라운드 설정 여부에 따라 1회성 미션으로 운영할 수 있습니다.

| Field | Value  | Example |
| ----- | ------ | ------- |
| 미션 타입 | SINGLE | SINGLE  |
| 라운드   | 미설정    | 1       |
| 달성 목표 | 달성 횟수  | 1       |

### 동작 방식

* 목표 달성 시 즉시 완료
* 반복 참여 불가

## SINGLE - 반복 목표 미션

* 동일 행동을 회차별로 반복 유도하고자 할 때 사용합니다.
* 라운드 설정에 따라 반복 미션으로 운영됩니다.

| Field | Value   | Example |
| ----- | ------- | ------- |
| 미션 타입 | SINGLE  | SINGLE  |
| 라운드   | 2 이상 설정 | 2       |
| 달성 목표 | 달성 횟수   | 1       |

### 동작 방식

| 라운드 | 목표 | 달성 | 상태 |
| --- | -- | -- | -- |
| 1회차 | 1  | 1  | 완료 |
| 2회차 | 1  | 1  | 완료 |

## ACCUMULATE - 누적 달성 미션

* 목표 값은 고정되며, 달성 수치는 회차가 바뀌어도 누적됩니다.
* 누적 수치가 목표에 도달하면 보상 지급됩니다.

| Field | Value      | Example    |
| ----- | ---------- | ---------- |
| 미션 타입 | ACCUMULATE | ACCUMULATE |
| 라운드   | 2 이상 설정    | 3          |
| 달성 목표 | 누적 달성 횟수   | 10         |

### 동작 방식

⚠️ 회차 변경 시 **달성 수치 초기화 없음**

| 라운드 | 목표 | 달성 | 상태        |
| --- | -- | -- | --------- |
| 1회차 | 10 | 4  | 4/10, 진행중 |
| 2회차 | 10 | 7  | 7/10, 진행중 |
| 3회차 | 10 | 10 | 10/10, 완료 |

## DIFFICULTY — 난이도 상승 미션

* 회차가 증가할수록 달성 목표의 난이도가 상승합니다.
* 회차 수와 난이도 설정 개수가 맞지 않으면 초기 난이도로 리셋됩니다.

⚠️ 회차 수와 난이도 설정 개수는 동일하게 설정하는 것을 권장합니다.

| Field | Value      | Example    |
| ----- | ---------- | ---------- |
| 미션 타입 | DIFFICULTY | DIFFICULTY |
| 라운드   | 2 이상 설정    | 4          |
| 달성 목표 | 난이도 단계     | 20,40,60   |

### 동작 방식

| 라운드 | 목표                                                 | 달성 | 상태         |
| --- | -------------------------------------------------- | -- | ---------- |
| 1회차 | 20                                                 | 10 | 10/20, 진행중 |
| 2회차 | 40                                                 | 5  | 5/40, 진행중  |
| 3회차 | 60                                                 | 40 | 40/60, 진행중 |
| 4회차 | <p>20<br>⚠️ 회차 수와 난이도 설정 개수가 맞지 않으면 초기 난이도로 리셋</p> | 20 | 20/20, 완료  |

## CHALLENGE - 컬렉션 미션

* 여러 개의 미션을 하나의 세트로 묶어 운영하는 미션입니다.
* 모든 미션을 달성해야 보상이 지급됩니다.

| Field | Value     | Example   |
| ----- | --------- | --------- |
| 미션 타입 | CHALLENGE | CHALLENGE |
| 라운드   | 미설정       | 1         |
| 달성 목표 | 연관 미션 ID  | 1,2,3     |

### 동작 방식

#### 미달성 예시

| 라운드          | 목표         | 달성 | 상태      |
| ------------ | ---------- | -- | ------- |
| 미션 1         | 1          | 1  | 완료      |
| 미션 2         | 1          | 1  | 완료      |
| 미션 3         | 1          | 0  | 진행중     |
| CHALLENGE 미션 | 미션1,2,3 달성 | 0  | **진행중** |

#### 달성 예시

| 라운드          | 목표         | 달성 | 상태     |
| ------------ | ---------- | -- | ------ |
| 미션 1         | 1          | 1  | 완료     |
| 미션 2         | 1          | 1  | 완료     |
| 미션 3         | 1          | 1  | 완료     |
| CHALLENGE 미션 | 미션1,2,3 달성 | 1  | **완료** |

## ⚠️ 운영자 주의사항

* 미션 타입은 등록 후 변경할 수 없습니다.
* 진행 중인 미션은 수정할 수 없습니다.
* 라운드 및 목표 설정이 일치하지 않을 경우, 예상과 다른 동작이 발생할 수 있습니다.
