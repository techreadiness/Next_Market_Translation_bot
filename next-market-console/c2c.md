# C2C 아이템 능력 관리

**아이템 능력 관리 페이지는 C2C 아이템에 사용할 ‘능력 항목(종류)’을 등록·관리하는 페이지입니다.**\
능력의 이름/표시 단위 등 기준 정보를 사전에 등록한 뒤, 아이템 등록 또는 C2C 판매 등록 시 해당 능력을 선택하여 사용할 수 있습니다.

제공 기능

* [아이템 능력 설정 등록](https://next-market-admin.gitbook.io/next-market-docs/api/undefined-2#post-api-v1-item-ability)
* [아이템 능력 설정 수정](https://next-market-admin.gitbook.io/next-market-docs/api/undefined-2#put-api-v1-item-ability-code)

## C2C 아이템 능력 등록 (개별 등록)

C2C 아이템 능력 등록은 **개별 등록 방식**을 지원합니다.\
소량의 아이템 능력을 등록하는 경우, 개별 등록을 통해 아이템 능력을 생성할 수 있습니다.

{% stepper %}
{% step %}
### C2C 아이템 능력 관리 페이지 선택

* NEXT Market Console 접속
* 좌측 메뉴에서 **C2C 아이템 능력 관리** 페이지 선택
{% endstep %}

{% step %}
### 능력 등록 선택

* 능력 등록 버튼을 선택하여 아이템 능력 등록 화면으로 이동합니다.

## C2C 아이템 능력 등록 (벌크 등록)

**다수의 아이템 능력을 한 번에 등록해야 하는 경우**, CSV 파일 업로드를 통해 아이템 능력을 일괄 등록할 수 있습니다.

{% stepper %}
{% step %}
### C2C 아이템 능력 관리 페이지 선택

* NEXT Market Console 접속
* 좌측 메뉴에서 C2C **아이템 능력 관리** 페이지 선택
{% endstep %}

{% step %}
### 템플릿 다운로드

* **템플릿 다운로드** 버튼을 선택하여 아이템 능력 벌크 등록용 **CSV 템플릿 파일**을 다운로드합니다.
* 템플릿에는 아이템 능력 등록에 필요한 기본 컬럼이 포함되어 있습니다.

<figure><img src="../.gitbook/assets/아이템 능력 관리_템플릿.png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
### 아이템 능력 리스트 작성하기

* 다운로드한 **CSV 템플릿 파일**에 등록할 아이템 능력 정보를 입력합니다.
* 각 항목의 작성 방법은 **아이템 능력 등록 (개별 등록)** 가이드를 참고하여 작성해 주세요.

⚠️ 필수 항목이 누락되거나 형식이 맞지 않을 경우 업로드가 실패할 수 있습니다.

<figure><img src="../.gitbook/assets/아이템 능력 관리_템플릿 엑셀.png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
### CSV 업로드

* 작성이 완료된 CSV 파일을 선택하여 업로드합니다.
* 파일 선택 후 **업로드** 버튼을 클릭하면 아이템 능력 벌크 등록이 진행됩니다.

#### ⚠️ 벌크 등록 시 유의사항

* CSV 업로드는 **신규 아이템 능력 또는 기존 등록 정보 수정** 용도로 사용됩니다.
* CSV 업로드 등록 전, **소량 테스트 업로드**를 권장합니다.

<figure><img src="../.gitbook/assets/아이템 능력 관리_CSV 업로드.png" alt=""><figcaption></figcaption></figure>
{% endstep %}
{% endstepper %}

## C2C 아이템 능력 수정

**이미 등록된 능력의 정보를 변경해야 하는 경우,** **개별 수정**과 **벌크 수정** 방식을 통해 수정할 수 있습니다.

### C2C 아이템 능력 수정 방식

#### 개별 수정

개별 아이템 능력의 정보를 수정하는 경우 사용합니다.

* **NEXT Market Console** 접속
* 좌측 메뉴에서 **C2C 아이템 능력 관리** 페이지 선택
* 아이템 능력 리스트에서 수정할 능력 선택
* **수정하기** 버튼 클릭
* 아이템 정보 변경 후 저장

<figure><img src="../.gitbook/assets/아이템 능력 관리_수정.png" alt=""><figcaption></figcaption></figure>

#### 벌크 수정

여러 아이템 능력의 정보를 한 번에 수정해야 하는 경우 사용합니다.

* **C2C 아이템 능력 관리** 페이지에서 **CSV 다운로드** 버튼 선택
* 다운로드한 CSV 파일에서 수정할 아이템 능력 정보 변경
* 수정이 완료된 CSV 파일 업로드
* 업로드 완료 후 수정 결과 확인

<figure><img src="../.gitbook/assets/C2C 아이템 능력 관리_업로드.png" alt=""><figcaption></figcaption></figure>

### ⚠️  수정 시 유의사항

* 아이템 능력 등록은 LINE NEXT와 사전 협의 후 진행해 주시기 바랍니다.
* 판매 중인 아이템의 정보 변경은 서비스 운영에 영향을 줄 수 있습니다.
* 벌크 수정 전에는 **소량 테스트 수정**을 권장합니다.
