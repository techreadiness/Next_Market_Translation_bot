# 미션 관리

미션 관리 페이지에서는 미션을 생성하고 운영 상태를 관리할 수 있습니다.

* 제공 기능
  * [미션 생성](https://next-market-admin.gitbook.io/next-market-docs/api/undefined-7#post-api-v1-incentive-mission)
  * [미션 수정](https://next-market-admin.gitbook.io/next-market-docs/api/undefined-7#put-api-v1-incentive-mission-missionid)

⚠️ 유의사항

* 미션 등록은 LINE NEXT와 사전 협의 후 진행해 주시기 바랍니다.
* 미션 관리는 CSV 업로드 기능을 지원하지 않습니다. 모든 미션은 Console을 통해 개별 등록해야 합니다.
* 미션 등록의 경우 NEXT Market에서만 노출되며, Dapp Portal USDt 리워드 미션에 노출되기를 희망하신다면 NEXT 사업 담당자에게 별도로 요청해 주시기 바랍니다.
  * 단, Dapp Portal USDt 리워드 노출 시, Logo 이미지 (420x420px)를 별도로 제출해야 합니다.

| ---------- | -------- | -------------------------------------------------------------------------------------------------------------------- |

{% step %}
### 미디어 정보 입력

* 미션이 보상타입이 아이템인 경우에만 등록합니다.

⚠️ USDt 보상의 경우 이미지 업로드는 필요하지 않습니다.

* 업로드 방식
  * URL 입력: 이미 업로드된 미디어 파일의 URL을 입력
  * 파일 업로드: 로컬 파일 선택 후 업로드
* 지원형식
  * IMAGE: JPG, PNG, WebP
* 업로드 제한
  * 파일 크기: 10MB 이하
  * 이미지 사이즈: 배경 포함 72x72px

⚠️ VIDEO 형식 업로드는 기능상 선택은 가능하나, **공식적으로 지원하지 않으므로 IMAGE 형식으로만 등록해 주시기 바랍니다.**

### 국가 제어 입력

* 미션의 국가별 접근 제한 정책을 설정합니다.
*   제한 국가 접근 옵션 선택

## 미션 수정

미션 수정은 **이미 등록된 미션의 정보를 변경해야 하는 경우** 사용할 수 있는 기능입니다.

* **NEXT Market Console** 접속
* 좌측 메뉴에서 **미션 관리** 페이지 선택
* 미션 정보 리스트에서 수정할 미션 선택
* **수정하기** 버튼 클릭
* 미션 정보 변경 후 저장

<figure><img src="../../../.gitbook/assets/미션 관리_수정.png" alt=""><figcaption></figcaption></figure>

### ⚠️  수정 시 유의사항

* 진행 중인 미션은 수정할 수 없습니다.
* 미션 시작일이 도래하지 않는 경우 수정은 가능하나, 미션 타입은 수정할 수 없습니다.
  * 미션 타입 변경이 필요한 경우, 신규 미션을 등록해야 합니다.
