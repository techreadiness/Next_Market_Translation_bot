# NEXT Market

## NEXT Market

* 게임 개발사와 Web2 게이머에게 새로운 자유 경제의 경험을 제공하는 글로벌 크로스보더 게임 마켓입니다.
* 게이머는 게임과의 1:1 계정연동을 통하여 인게임에서 손쉽게 아이템을 거래하고, 게임 개발사는 글로벌 스테이블 코인 기반의 수익화를 실현할 수 있습니다.

## LINE NEXT

* Web3 디지털 자산 생태계 구축을 목표로 NEXT Market, Mini Dapp, LINE/Web 기반의 Web3 Wallet 서비스 제공
* 8개월 간 6,500만개의 액티브 월렛과 160만개의 구매 월렛을 확보 → 아시아 최대 규모의 월렛 네트워크 보유
* 아시아 디지털 달러 게이트웨이를 위한 테더와의 전략적 파트너십 체결

## NEXT Market 주요 기능

### 1. 게임 임베드 마켓

게이머들은 별도의 로그인 없이 게임 계정으로 인게임 내에서 바로 마켓에 접근하고 인게임 내에서 아이템 거래를 할 수 있습니다.

### 2. 다중 통화 웹샵

전세계 로컬 페이먼트와 스테이블 코인(USDt) 결제를 지원하는 커스텀 웹샵 환경을 제공합니다.

### 3. 크로스보더 마켓플레이스

* 게이머들은 인게임 계정만으로 B2C/C2C 아이템 거래가 가능합니다.
* 구매자
  * 로컬 페이먼트 또는 스테이블 코인으로 결제
  * 지갑 개설 없이 인게임 메일박스에서 아이템 수령 가능
* 판매자
  * 수수료 없이 스테이블 코인으로 즉시 정산
  * 소셜ID와 OKX, Bitget Wallet으로 로그인 후 정산 가능

### 4. 스테이블 코인 리워드 시스템

* 미션 및 플레이 기반으로 스테이블 코인 리워드 지급
* 게이머는 플레이 -> 스테이블 코인 리워드 획득 → 클레임 흐름을 통해 게임 내 경제에 참여
* 보상 받은 스테이블 코인으로 재구매 또는 추가 이익 획득 → 재투자 순환 구조 기여
  * 스테이블 코인 보상 받은 게이머 대상 재구매 비중 90% 이상 기록

## 개발사 온보딩 지원

### 1. 마케팅 지원

* 게임 마케팅 리워드 지원금 제공 (온보딩 조건에 따라 상이)
* LINE의 Dapp Portal 내 게임 리워드 Featured 노출
* LINE NEXT의 공식 OA에서 메시지 발송 지원

### 2. 비즈니스 지원

* 매출 통계 및 분석 대시보드 제공
* 유저 리텐션 인사이트 제공
