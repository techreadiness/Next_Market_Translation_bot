# Glossary



<table><thead><tr><th width="189.46875">Terms</th><th>Description</th><th>Notes</th></tr></thead><tbody><tr><td>Game</td><td>Game developer&#x27;s server/client</td><td></td></tr><tr><td>NEXT Market, Store</td><td>Line Next Server/Client</td><td></td></tr><tr><td>Preview</td><td>Preview environment with restricted access for general users</td><td></td></tr><tr><td>Prod</td><td>Environment where actual services are integrated (Production)</td><td></td></tr><tr><td>guid</td><td>A unique identifier linking Game, NEXT Market, and Store. Since it is exposed externally, it must be defined as a value that is difficult to infer and not vulnerable to security risks. Used as an identifier for authentication<br>during purchases on NEXT Market and for requesting item delivery after purchase.</td><td>When a guid is transferred to another user in game features like &#x27;Account Gifting&#x27;, the user&#x27;s personal information is also transferred as they are the authenticated user in the store. Therefore, the guid must be managed as the user&#x27;s personal information.</td></tr><tr><td>SKU<br>(Stock Keeping Unit)</td><td>A unique item code assigned to goods shipped to the game to identify and manage inventory (a value defined by the game).</td><td></td></tr><tr><td>Sealing</td><td>Registering in-game items for sale on the C2C Market</td><td></td></tr><tr><td>Unsealing</td><td>Canceling the sale registration of an item for C2C and restoring it to the in-game account</td><td></td></tr><tr><td>Section</td><td>A list unit for displaying B2C products. When only one is set, it displays as a single column. When two or more are set, the top tab and anchor functionality activates.</td><td></td></tr></tbody></table>
