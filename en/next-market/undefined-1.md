# Onboarding Process

### Diagram

<figure><img src="../../.gitbook/assets/NEXTMarket_Onboarding.png" alt=""><figcaption></figcaption></figure>### Description

1. LINE NEXT and the game developer (or publisher) negotiate and sign a contract for NEXT Market integration.
2. A dedicated communication hotline channel is established between NEXT Market and the game developer (or publisher) for rapid integration.
3. Technical specifications for NEXT Market integration are shared, and the game developer (or publisher) reviews them.
4. If technical support or development assistance for new specifications is needed, both parties can discuss this through meetings.
5. The game developer (or publisher) submits the information required for NEXT Market registration.
6. NEXT Market configures the environment and store based on the received information.
7. The game developer (or publisher) receives the necessary API permissions for integration and proceeds with development.
