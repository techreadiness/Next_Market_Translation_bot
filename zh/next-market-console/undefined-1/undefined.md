# 任务类型设置指南

任务的运作方式与奖励发放机制会根据任务类型、回合数（轮次）及达成目标的设置而有所不同。
请参考以下示例，选择符合目标需求的任务类型。

## SINGLE - 单一目标任务

* 完成单一目标即可获得奖励。
* 可根据是否设置回合数，作为一次性任务运行。

| 字段        | 值         | 示例        |
| ----- | ----- | ----- |
| 任务类型 | SINGLE | SINGLE      |
| 回合        | 未设置    | 1        |
| 达成目标 | 达成次数 | 1        |

### 运行机制

* 目标达成即刻完成
* 不可重复参与

## SINGLE - 重复目标任务

* 用于引导用户按回合重复相同行为。
* 根据回合设置可作为重复任务运行。

| 字段 | 值       | 示例 |
| ----- | ------- | ------- |
| 任务类型 | SINGLE | SINGLE |
| 回合数   | 设置2次以上 | 2       |
| 达成目标 | 达成次数   | 1       |

### 运作方式

| 回合   | 目标   | 达成   | 状态   |
| --- | -- | -- | -- |
| 第1回合 | 1  | 1  | 完成  |
| 第2轮 | 1      | 1      | 完成 |

## 累积达成任务

* 目标值固定，达成数值随轮次变化持续累积。
* 累积数值达到目标时发放奖励。

| 字段        | 数值        | 示例        |
| ----- | ---------- | ---------- |
| 任务类型 | 累积型        | 累积型        |
| 回合数     | 设置2回合及以上 | 3          |
| 达成目标 | 累计达成次数   | 10         |

### 运作机制

⚠️ 更换回合时**达成数值不归零**

| 回合数 | 目标        | 达成        | 状态        |
| --- | -- | -- | --------- |
| 第1轮 | 10 | 4  | 4/10, 进行中 |
| 第2轮 | 10 | 7  | 7/10, 进行中 |
| 第3轮 | 10 | 10 | 10/10, 完成 |

## DIFFICULTY — 难度递增任务

* 随着回合数增加，达成目标的难度将逐步提升。
* 若回合数与难度设置数量不匹配，将重置为初始难度。

⚠️ 建议将回合数与难度设置数量保持一致。

| 字段 | 数值      | 示例    |
| ----- | ---------- | ---------- |
| 任务类型 | 难度等级 | 难度等级 |
| 回合数    | 2次以上设置 | 4          |
| 达成目标 | 难度阶段     | 20,40,60   |

### 运作机制

| 回合数 | 目标                                                 | 达成 | 状态         |
| --- | -------------------------------------------------- | -- | ---------- |
| 第1轮 | 20                                                 | 10 | 10/20, 进行中 |
| 第2轮 | 40                                                 | 5  | 5/40, 进行中  |
| 第3轮 | 60                                                 | 40 | 40/60, 进行中 |
| 第4轮 | <p>20<br>⚠️ 若回合数与难度设置数量不匹配，将重置为初始难度</p> | 20 | 20/20, 已完成  |

## CHALLENGE - 系列任务

* 将多个任务组合为一套进行运营的任务类型。
* 需达成所有任务方可获得奖励。

| 字段 | 数值     | 示例   |
| ----- | --------- | --------- |
| 任务类型 | CHALLENGE | CHALLENGE |
| 回合数      | 未设定       | 1         |
| 达成目标 | 关联任务ID  | 1,2,3     |

### 运作机制

#### 未达成示例

| 回合数        | 目标         | 达成 | 状态      |
| ------------ | ---------- | -- | ------- |
| 任务1         | 1          | 1  | 已完成      |
| 任务2         | 1          | 1  | 已完成      |
| 任务3         | 1          | 0  | 进行中     |
| CHALLENGE 任务 | 任务1,2,3达成 | 0  | **进行中** |

#### 达成示例

| 回合          | 目标         | 达成 | 状态     |
| ------------ | ---------- | -- | ------ |
| 任务1         | 1          | 1  | 完成     |
| 任务2         | 1          | 1  | 完成     |
| 任务3         | 1          | 1  | 完成     |
| 挑战任务         | 任务1,2,3达成         | 1  | **完成**         |

## ⚠️ 运营者注意事项

* 任务类型注册后不可更改。
* 进行中的任务不可修改。
* 若回合数与目标设置不匹配，可能导致与预期不符的运行结果。
