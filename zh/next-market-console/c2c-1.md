# C2C预设管理

**C2C预设管理页面用于注册和管理C2C商品使用的分类（商品分类层级结构）。**\
需预先注册C2C商品的分类信息，在C2C商品上架时可选择对应分类进行使用。\
分类设置将同步至Marketplace的筛选区域，商品将按分类进行展示。

\## C2C预设注册

C2C商品分类设置最多支持2级深度：

* 1级：父级预设
* 2级：子级预设

{% stepper %}
{% step %}
#### 选择C2C预设管理页面

* 登录NEXT Market控制台
* 在左侧菜单选择**C2C预设管理**页面
{% endstep %}

{% step %}
#### 选择创建预设

* 点击创建预设按钮进入注册界面
{% endstep %}

{% step %}
#### 基础信息输入

填写预设（类别）的基础信息。

| 字段        | 值                                                                                                                                                                                    |
| --------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| 等级        | <ul><li><p>预设可设置为1~2级</p><ul><li>级别1为顶级类别，不可设置父级预设</li><li>级别2将作为级别1的子类别进行注册</li></ul></li><li><p>例）</p><ul><li>级别1：武器</li><li>级别2：弓、大剑、法杖、弩、匕首</li></ul></li></ul>                  |
| 父级预设      | <ul><li><p>注册二级预设时，需设置父级预设（一级）</p><ul><li>1级预设无法设置父级预设</li></ul></li><li><p>例）</p><ul><li>预设：弓</li><li>父级预设：武器</li></ul></li></ul>                                                   |
| 颜色        | <ul><li>设置在类别中显示的颜色</li><li>可选颜色：蓝色、黄色、红色、绿色</li><li>选择无时应用默认颜色</li></ul><p>⚠️ 建议仅在等级（稀有度）分类中设置颜色。</p><ul><li><p>例）</p><ul><li>传说：黄色</li><li>稀有：蓝色</li><li>高级：绿色</li></ul></li></ul> |
| 预设名称（多语言） | <ul><li>多语言输入支持：en, ko, ja, th, zh</li><li><p>交易所内物品详情页显示的数值</p><ul><li>必须注册市场提供的语言</li></ul></li></ul>                                                                              |
{% endstep %}

{% step %}
#### 物品注册

预设包含的物品（SKU）设置并非在初始注册时进行，而需通过注册完成后的修改功能完成。
{% endstep %}
{% endstepper %}

## C2C预设修改

**已注册预设的信息可进行修改。**

* 登录**NEXT Market控制台**
* 在左侧菜单选择**C2C预设管理**页面
* 从预设列表中选择待修改的预设
* 点击**修改**按钮
* 修改预设信息后保存

\### ⚠️ 修改注意事项

* 预设注册需事先与LINE NEXT协商确认
* 预设等级不可修改。如需变更等级，请注册新预设
* 已运营预设的信息变更可能影响服务运行
