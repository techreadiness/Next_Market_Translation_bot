# B2C商品管理

在B2C商品管理页面，您可以注册和管理B2C商品。

该页面通过Console环境提供与B2C商品相关的API功能。

* 提供功能
  * [版块注册](https://next-market-admin.gitbook.io/next-market-docs/api/undefined-3#post-api-v1-sale-b2c-section)
  * [分区变更](https://next-market-admin.gitbook.io/next-market-docs/api/undefined-3#patch-api-v1-sale-b2c-section-sectionid)
  * [销售注册](https://next-market-admin.gitbook.io/next-market-docs/api/undefined-4#post-api-v1-sale-b2c)
  * [销售变更](https://next-market-admin.gitbook.io/next-market-docs/api/undefined-4#patch-api-v1-sale-b2c-b2csaleid)

⚠️ 注意事项

* B2C商品注册不支持CSV批量上传功能。所有B2C商品均需通过Console单独注册。
* 注册B2C商品前，请务必与LINE NEXT进行事前协商。

## 版块管理

版块是**用于按商品组分类展示B2C商品的分类区域**。 当商品种类较多或需区分不同商品时，可利用版块功能。

### 分区设置规范

* 可展示分区数量：**最少2个～最多4个**
* 为保障界面稳定性，**上限限定为4个**
* 若无需分区管理，**可不设置分区**

* 登录NEXT Market控制台
* 在左侧菜单选择**B2C商品管理**页面
{% endstep %}

{% step %}
#### 分区注册

* 在分区管理区域选择"添加分区"按钮
* 跳转至分区注册界面
{% endstep %}

## B2C商品注册

B2C商品注册功能可创建B2C商品并管理销售信息。 商品通过基础信息、价格、购买限制、销售期限及关联商品信息输入构成。

{% stepper %}
{% step %}
#### 选择B2C商品管理页面

* 登录NEXT Market控制台
* 在左侧菜单选择**B2C商品管理**页面
{% endstep %}

{% step %}
#### 商品注册

* 点击商品注册按钮进入注册界面
{% endstep %}

{% step %}

## B2C商品修改

B2C商品修改功能适用于**需变更已注册商品信息**的情形。

* 登录**NEXT Market控制台**
* 左侧菜单选择**B2C商品管理**页面
* 在商品信息列表中选定待修改商品
* 点击**修改**按钮
* 修改商品信息后保存

\### ⚠️ 修改注意事项

* 销售中商品的信息变更可能影响服务运营，建议修改前充分确认。
* 与价格信息关联的商品清单（SKU）属于不可修改项。
  * 如需修改，请先下架原商品，再以新商品形式重新上架。
* 对正在销售或已售罄的商品，可能影响购买记录的关键项不可修改。
  * 不可修改项：
    * 销售开始日期
    * 总购买限制数量
