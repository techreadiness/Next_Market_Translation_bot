# NEXT Market

## NEXT Market

* ゲーム開発者とWeb2ゲーマーに新たな自由経済の体験を提供するグローバルクロスボーダーゲームマーケットです。
* ゲーマーはゲームとの1:1アカウント連携により、ゲーム内で手軽にアイテムを取引でき、ゲーム開発者はグローバルステーブルコイン基盤の収益化を実現できます。

<figure><img src=".gitbook/assets/NEXT Market_2.png" alt=""><figcaption></figcaption></figure>## NEXT Market 主要機能

### 1. ゲーム埋め込みマーケット

ゲーマーは別途ログイン不要でゲームアカウントからインゲーム内で直接マーケットにアクセスし、インゲーム内でアイテム取引が可能です。

### 2. マルチ通貨ウェブショップ

全世界のローカルペイメントとステーブルコイン(USDt)決済をサポートするカスタムウェブショップ環境を提供。

### 3. クロスボーダーマーケットプレイス

* ゲーマーはゲーム内アカウントのみでB2C/C2Cアイテム取引が可能
* 購入者
  * ローカル決済またはステーブルコインで決済
  * ウォレット開設不要でゲーム内メールボックスからアイテム受取可能
* 販売者
  * 手数料なしでステーブルコインによる即時決済
  * ソーシャルIDとOKX、Bitget Walletでログイン後、決済可能

### 4. ステーブルコイン報酬システム

* ミッションおよびプレイベースでステーブルコイン報酬を支給
* ゲーマーはプレイ→ステーブルコイン報酬獲得→クレームの流れでゲーム内経済に参加
* 報酬として受け取ったステーブルコインで再購入または追加利益獲得→再投資循環構造に貢献
  * ステーブルコイン報酬を受けたゲーマーの再購入割合は90%以上を記録
