# C2Cアイテム能力管理

**アイテム能力管理ページは、C2Cアイテムに使用する「能力項目（種類）」を登録・管理するページです。**\
能力の名前/表示単位などの基準情報を事前に登録した後、アイテム登録またはC2C販売登録時に該当能力を選択して使用できます。

提供機能

* [アイテム能力設定登録](https://next-market-admin.gitbook.io/next-market-docs/api/undefined-2#post-api-v1-item-ability)
* [アイテム能力設定修正](https://next-market-admin.gitbook.io/next-market-docs/api/undefined-2#put-api-v1-item-ability-code)

## C2Cアイテム能力登録（個別登録）

C2Cアイテム能力登録は**個別登録方式**をサポートします。
少量のアイテム能力を登録する場合、個別登録を通じてアイテム能力を生成できます。

{% stepper %}
{% step %}
### C2Cアイテム能力管理ページの選択

* NEXT Market Console にアクセス
* 左メニューから**C2Cアイテム能力管理**ページを選択
{% endstep %}

{% step %}
### 能力登録の選択

* 能力登録ボタンを選択し、アイテム能力登録画面に移動します。

{% endstep %}

## C2Cアイテム能力登録（一括登録）

**複数のアイテム能力を一度に登録する必要がある場合**、CSVファイルアップロードを通じてアイテム能力を一括登録できます。

{% stepper %}
{% step %}
### C2Cアイテム能力管理ページの選択

* NEXT Market Consoleにアクセス
* 左メニューからC2C **アイテム能力管理**ページを選択
{% endstep %}

{% step %}
### テンプレートダウンロード

* **テンプレートダウンロード**ボタンを選択し、アイテム能力一括登録用**CSVテンプレートファイル**をダウンロードします。
* テンプレートにはアイテム能力登録に必要な基本カラムが含まれています。

<figure><img src="../.gitbook/assets/아이템 능력 관리_템플릿.png" alt=""><figcaption></figcaption></figure>{% endstep %}

{% step %}
### アイテム能力リストの作成

* ダウンロードした**CSVテンプレートファイル**に登録するアイテム能力情報を入力します。
* 各項目の作成方法は**アイテム能力登録（個別登録）**ガイドを参照して作成してください。

⚠️ 必須項目が欠落している場合や形式が正しくない場合、アップロードが失敗する可能性があります。

<figure><img src="../.gitbook/assets/아이템 능력 관리_템플릿 엑셀.png" alt=""><figcaption></figcaption></figure>{% endstep %}

{% step %}
### CSVアップロード

* 作成が完了したCSVファイルを選択してアップロードします。
* ファイル選択後、**アップロード**ボタンをクリックするとアイテム能力の一括登録が進行します。

#### ⚠️ 一括登録時の注意事項

* CSVアップロードは**新規アイテム能力または既存登録情報の修正**用途で使用されます。
* CSVアップロード登録前には、**少量のテストアップロード**を推奨します。

<figure><img src="../.gitbook/assets/아이템 능력 관리_CSV 업로드.png" alt=""><figcaption></figcaption></figure>{% endstep %}
{% endstepper %}

## C2Cアイテム能力の修正

**既に登録された能力の情報を変更する必要がある場合、** **個別修正**と**一括修正**の方法で修正できます。

### C2Cアイテム能力の修正方法

#### 個別修正

個別アイテム能力の情報を修正する場合に使用します。

* **NEXT Market Console** にアクセス
* 左メニューから **C2Cアイテム能力管理** ページを選択
* アイテム能力リストから修正する能力を選択
* **修正する** ボタンをクリック
* アイテム情報を変更後、保存

<figure><img src="../.gitbook/assets/아이템 능력 관리_수정.png" alt=""><figcaption></figcaption></figure>#### 一括修正

複数のアイテム能力情報を一度に修正する必要がある場合に使用します。

* **C2Cアイテム能力管理**ページで**CSVダウンロード**ボタンを選択
* ダウンロードしたCSVファイルで修正するアイテム能力情報を変更
* 修正完了したCSVファイルをアップロード
* アップロード完了後、修正結果を確認

<figure><img src="../.gitbook/assets/C2C 아이템 능력 관리_업로드.png" alt=""><figcaption></figcaption></figure>### ⚠️ 修正時の注意事項

* アイテム能力の登録は、LINE NEXTと事前に協議の上、行ってください。
* 販売中のアイテムの情報変更は、サービス運営に影響を与える可能性があります。
* 一括修正前には、**少量のテスト修正**を推奨します。
