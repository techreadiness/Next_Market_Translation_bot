# アイテム管理

アイテム管理ページでは、B2CおよびC2C取引に使用されるアイテムを登録・管理できます。

該当ページを通じて、アイテム登録API機能をConsole環境で提供します。

* 提供機能
  * [メディアファイルアップロード](https://next-market-admin.gitbook.io/next-market-docs/api/undefined-2#post-api-v1-media-upload)
  * [アイテム登録](https://next-market-admin.gitbook.io/next-market-docs/api/undefined-2#post-api-v1-item)
  * [アイテム変更](https://next-market-admin.gitbook.io/next-market-docs/api/undefined-2#put-api-v1-item-sku)

## アイテム登録 (個別登録)

アイテム登録は**個別登録方式**をサポートします。
少量のアイテムを登録する場合、個別登録を通じてアイテムを生成できます。

{% stepper %}
{% step %}
### アイテム管理ページの選択

* NEXT Market Console にアクセス
* 左メニューから**アイテム管理**ページを選択
{% endstep %}

{% step %}
### アイテム登録の選択

* アイテム登録ボタンを選択し、アイテム登録画面に移動します。

<figure><img src="../../.gitbook/assets/아이템 관리_아이템 등록.png" alt=""><figcaption></figcaption></figure>{% endstep %}

{% step %}
### メディアファイルのアップロード

* アイテムで使用するメディアファイルをアップロードします。

#### アップロード方式

* URL入力：既にアップロード済みのメディアファイルのURLを入力
* ファイルアップロード：ローカルファイルを選択後アップロード

#### 対応形式

* IMAGE: JPG, PNG, WebP

⚠️ VIDEO形式のアップロードは機能上選択可能ですが、**公式にはサポートしていないため、IMAGE形式でのみ登録してください。**

#### アップロード制限

* ファイルサイズ: 10MB以下
* 画像サイズ: 背景を含む72x72px

<figure><img src="../../.gitbook/assets/아이템 관리_미디어 파일 업로드.png" alt=""><figcaption></figcaption></figure>
{% endstep %}

{% step %}
### 基本情報入力

アイテムの基本情報を入力します。

| フィールド             | 値                                                                                                                                                                                                                                                              |
| ----------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| SKU (必須)    | <ul><li>アイテムを識別・管理するための固有品目コード</li><li>重複登録不可</li><li><p>形式：200文字未満のユーザーが推測可能な値で、英字、数字、特殊文字使用可</p><ul><li>特殊文字はアンダースコア <code>_</code> 使用不可のためハイフン <code>-</code>  の使用を推奨</li></ul><p>⚠️ SKUは実際のゲームで使用するアイテム識別子と同じ値で入力してください。</p></li></ul> |
| アイテム名 (必須) | <ul><li>多言語入力対応: en, ko, ja, th, zh</li><li><p>アイテムリストおよび詳細画面に表示される値</p><ul><li>マーケットで提供される言語については必ず登録が必要</li></ul></li></ul>                                                                                                                                  |
| アイテム説明 (必須) | <ul><li>アイテム登録ページではアイテム名として設定</li><li>詳細説明が必要な場合、<strong>CSVアップ</strong>ロード方式で別途登録が必要</li></ul>                                                                                                                                                        |

⚠️ C2C販売用アイテム登録時の能力は、アイテム(SKU)単位で入力する領域です。C2C取引時のアイテム能力表示や条件検証に活用できます。

| フィールド        | 値                                                                                                                                                                                                                                                                                                                                                                                               |
| ------------ | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 能力名        | <ul><li>多言語入力サポート: en, ko, ja, th, zh</li><li>能力表示名</li><li>マーケットで提供される言語については必ず登録が必要</li></ul>                                                                                                                                                                                                                                                                                                                         |

## アイテム登録（一括登録）

アイテム一括登録は、**複数のアイテムを一度に登録する必要がある場合**、CSVファイルのアップロードを通じてアイテムを一括登録できる機能です。

{% stepper %}
{% step %}
### アイテム管理ページの選択

* NEXT Market Consoleにアクセス
* 左メニューから**アイテム管理**ページを選択
{% endstep %}

{% step %}
### テンプレートダウンロード

* **テンプレートダウンロード**ボタンを選択し、アイテム一括登録用**CSVテンプレートファイル**をダウンロードします。
* テンプレートにはアイテム登録に必要な基本カラムが含まれています。

<figure><img src="../../.gitbook/assets/아이템 관리_템플릿.png" alt=""><figcaption></figcaption></figure>{% endstep %}

{% step %}
### アイテムリストの作成

* ダウンロードした**CSVテンプレートファイル**に登録するアイテム情報を入力します。
* 各項目の作成方法は、**アイテム登録（個別登録）**ガイドを参照して作成してください。
* ただし、imageUrlはファイルアップロードではなく、既にアップロードされたメディアファイルのURL入力方式のみをサポートします。

⚠️ 必須項目が欠落している場合や形式が正しくない場合、アップロードが失敗する可能性があります。

<figure><img src="../../.gitbook/assets/아이템 관리_csv 템플릿.png" alt=""><figcaption></figcaption></figure>{% endstep %}

{% step %}
### CSVアップロード

* 作成が完了したCSVファイルを選択してアップロードします。
* ファイル選択後、**アップロード**ボタンをクリックするとアイテムの一括登録が進行します。

#### ⚠️ 一括登録時の注意事項

* CSVアップロードは**新規アイテムまたは既存登録アイテム情報の修正**用途で使用されます。
* CSVアップロード登録前には、**少量のテストアップロード**を推奨します。

<figure><img src="../../.gitbook/assets/아이템 관리_csv 업로드.png" alt=""><figcaption></figcaption></figure>{% endstep %}
{% endstepper %}

## アイテム修正

アイテム修正は、**既に登録済みのアイテム情報を変更する必要がある場合**に使用できる機能です。
NEXT Market Consoleでは、アイテム修正のために**個別修正**と**一括修正**の2つの方法を提供しています。

### アイテム修正方式

#### 個別修正

個別アイテムの情報を修正する場合に使用します。

* **NEXT Market Console** にアクセス
* 左側メニューから**アイテム管理**ページを選択
* アイテムリストから修正するアイテムを選択
* **修正する**ボタンをクリック
* アイテム情報を変更後、保存

<figure><img src="../../.gitbook/assets/아이템 관리_수정.png" alt=""><figcaption></figcaption></figure>#### 一括編集

複数のアイテム情報を一度に編集する必要がある場合に使用します。

* **アイテム管理**ページで**CSVダウンロード**ボタンを選択
* ダウンロードしたCSVファイルで編集するアイテム情報を変更
* 編集完了したCSVファイルをアップロード
* アップロード完了後、編集結果を確認

<figure><img src="../../.gitbook/assets/아이템 관리_csv 다운로드.png" alt=""><figcaption></figcaption></figure>### ⚠️ 編集時の注意事項

* 販売中のアイテム情報変更はサービス運営に影響を与える可能性があります。
* 一括編集前には**少量のテスト編集**を推奨します。
